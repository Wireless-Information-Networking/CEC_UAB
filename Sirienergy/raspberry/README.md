# System Installation and Telemetry Guide

1. The initial phase requires obtaining the Venus OS software. Users can review the core installation procedures at the primary repository.
[Venus OS Repository](https://github.com/victronenergy/venus)

Finding the precise disk image that matches the target Raspberry Pi hardware is a necessary procedure. The official wiki supplies a comprehensive guide for identifying the correct version.
[Raspberry Pi Image Selection](https://github.com/victronenergy/venus/wiki/raspberrypi-install-venus-image)

2. The dbus flashmq application operates as a rapid digital interpreter for solar power installations. The program translates internal machine instructions generated by batteries and inverters into standard internet protocols. Such a translation enables dependable remote observation of real time power data. It also grants system administrators the ability to modify equipment configurations from distant locations. The official documentation provides complete operational details.
[Dbus Flashmq Documentation](https://github.com/victronenergy/dbus-flashmq/blob/master/README.md)

3. Establishing an active data link requires pointing the local MQTT client to the specific IP address of the SiriEnergy server. Configuring the edge device with this target IP ensures that the raw telemetry flows directly into the cloud backend. A successful network connection immediately pushes the live energy metrics to the web dashboard for remote visualization.
